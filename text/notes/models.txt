Для создания модели нужны следующие компоненты:

Сама модель, для которой выбраны количество слоёв, блоков, параметр Dropout.
Области обучения, валидации, тестирования: задаются пикселями healpix из разбиения 2.
Каталог обучающей выборки: список скоплений, по которому мы хотим обучить модель.
Координаты патчей для обучения: составляются по каталогу обучающей выборки. Среди всех пикселей 
из разбиения healpix 2^11 выбираются те, которые являются центрами патчей, в которые попадают 
какие-либо скопления из обучающей выборки.

Чтобы обучить модель, на каждой эпохе генератор выбирает случайным образом заданное число 
координат патчей, строит по ним сами изображения патчей и маски к ним.

Чтобы обучить модель unet_pz_only, в качестве обучающей выборки используется каталог planck_z.
После получения лучшего каталога по этой модели, выбираются скопления из каталога ACT, которые 
способна обнаружить данная модель (есть обнаружения в радиусе 5' с prediction index >= 0.1). 
Такие скопления заносятся в каталог act_found, и на основе его и каталога planck_z создается модель 
unet_pz_act_found.

Аналогично найденные моделью unet_pz_act_found скопления из ACT и MCXC заносятся в каталог 
all_found, и на его основе обучается модель unet_all_found.
